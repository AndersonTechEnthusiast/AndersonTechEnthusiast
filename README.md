**Automação Web & Browser**
- `Selenium` + `ChromeDriver` + `undetected-chromedriver` — automação de login, scraping, formulários governamentais, upload em redes sociais
- `PyAutoGUI` + `PyGetWindow` — controle de GUI, mouse, teclado e foco de janelas
- `BeautifulSoup` + `Requests` — extração e análise de conteúdo HTML
- `Instaloader` — download automatizado de conteúdo do Instagram com proteção anti-ban

**Processamento de Documentos & Mídia**
- `python-docx` + `ReportLab` + `PyMuPDF (fitz)` — criação, leitura e conversão de DOCX/PDF com fidelidade de formatação
- `Pillow (PIL)` — manipulação e extração de imagens
- `MoviePy` — edição de vídeo: redimensionamento, crop, concatenação, legendas animadas, sincronização de áudio
- `gTTS` — síntese de voz (text-to-speech) em português
- `pytubefix` — download de vídeos e áudios do YouTube
- `Whisper AI (OpenAI)` — transcrição automática de áudio para texto

**Dados, ML & Terminal**
- `scikit-learn` — modelos de Machine Learning (Árvore de Decisão, classificação, predição)
- `openpyxl` — geração e manipulação de planilhas Excel com formatação
- `colorama` + `pyfiglet` + `tqdm` — interfaces interativas e visuais no terminal
- `psutil` — monitoramento de sistema e detecção de unidades de armazenamento
- `tkinter` — interfaces gráficas desktop (GUI)
- `validators` + `re` — validação de dados, e-mails, URLs e expressões regulares
- `Unidecode` + `xml.etree.ElementTree` — normalização de texto e manipulação de XML
